# Data Challenge Telecom Paris — Face Occlusion Prediction

Two tracks are implemented in this repository:

| Track | Folder | Description |
|---|---|---|
| Model Training | [`model-training/`](model-training/) | Deep learning approach with EfficientNet, ConvNeXt and ViT ensembles |
| Training-Free | [`train-free/`](train-free/) | Computer vision baseline without model training |

See each folder's README for setup and usage instructions.
